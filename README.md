 # 자바스크립트 연산자
## 표현식
표현식expression : 리터럴(값), 식별자(변수), 연산자, 함수 호출등의 조합을
의미<br> 
표현식은 평가evaluate(표현식이 실행되어 결과값 생성)되어 하나의 값을 만듦

```javascript
    10; // 리터럴 표현식
    sum; // 식별자 표현식
    10 + 20; // 연산자 표현식
    Date(); // 함수 표현식
```
